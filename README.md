## Metalλαγμένος
My new go to renderer because MoltenVK is missing stuff.

## Build Instructions
### Prerequisites

Before you start, install **CMake**
### Build 
```bash
git clone https://github.com/Qirias/Metallagmenos
cd Metallagmenos
```
### Xcode
```bash
chmod +x xcode.sh
./xcode.sh
open build_xcode/Metallagmenos.xcodeproj
```

### Standalone
```bash
chmod +x standalone.sh
./standalone.sh
```
